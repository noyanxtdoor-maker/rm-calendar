# RM Calendar

RM Calendar is a mobile-first field productivity platform. The first release is a web beta designed for phone-sized use; native mobile apps will follow once the workflow is validated and a mobile release budget is available.

## Architecture

The project’s product and architecture specifications are in [`docs/`](docs/):

- Product vision and Phase 0 discovery
- Domain model and critical workflows
- Phase 1 information architecture

## Product direction

RM Calendar helps field workers plan activities, manage contacts and places, capture outcomes, and create dependable follow-ups—even when connectivity is limited.
